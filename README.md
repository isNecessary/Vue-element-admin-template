## 介绍
  在原来的vue-element-admin的基础上新增了一些功能，包括：
  1、支持多种导航栏，包括侧边导航栏、顶部导航栏、混合模式三种方式、且顶部导航栏超过宽度时自动折叠。
  2、支持修改主题色。
